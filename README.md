# CSV Parser 

Интерактивное desktop-приложение на Qt для просмотра и редактирования CSV-файлов. Позволяет быстро работать с табличными данными без громоздких офисных пакетов.

## Возможности
- Открытие CSV с произвольным разделителем (по умолчанию запятая).
- Отображение данных в `QTableWidget`, привычный табличный интерфейс.
- Редактирование ячеек в реальном времени с сохранением в модель.
- Добавление строк и столбцов прямо из графического интерфейса.
- Сохранение поверх исходного файла или экспорт в новый путь.
- Диалоговые подсказки и предупреждения при некорректных действиях.

## Стек технологий
- C++17
- Qt Widgets (QMainWindow, QTableWidget, QAction, QFileDialog)
- STL: fstream, vector, string

```
├─ mainwindow.* # GUI-логика
├─ csvparser.* # Классы Row и CSVFile

└─ cs.csv # Пример CSV
```

## Основные классы
- `Row` — хранит значения строки, управляет доступом к ячейкам и добавлением колонок.
- `CSVFile` — парсит и сохраняет CSV, умеет расширять таблицу при редактировании новых ячеек.
- `MainWindow` — связывает графический интерфейс с моделью данных и обрабатывает действия пользователя.
